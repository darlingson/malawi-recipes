Malawi Recipes Project

This project was inspired by the challenge of and subsiquite failure to cook Usipa.

The Malawi Recipes Project is a learning experience for three key technologies:

  • Flask: Used to build the backend of the project, Flask is a lightweight web framework in Python that allows building robust and scalable web applications.
  • SQLite: For efficient data management, SQLite is chosen as the database system. It's easy to set up and well-suited for smaller to medium-sized applications.
  • Tailwind CSS: Integrated for making the project visually appealing and responsive, Tailwind CSS provides a utility-first approach, making it easier to design and style the user interface.
